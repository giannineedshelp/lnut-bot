LANGUAGENUT BOT — COMMANDS LIST (v3)
========================
USER COMMANDS
========================

/login <username> <password>
- Log in to LanguageNut, encrypts and stores credentials
- Token auto-refreshes on expiry

/logout
- Remove stored credentials + token

/homework
- Lists all homework assignments with a paginated view
- Shows completion percentages and progress

/do
- Complete homework tasks using an interactive selector
- Choose specific assignments or "Do ALL"

/quick-do <task>
- Quick-complete a task by homework:index format
- Multi-task: 123:0, 456:2
- Uses concurrency setting to run in parallel

/settings
- Interactive panel with persistent per-guild settings
- Speed, Accuracy, Concurrency, Retry, Stealth toggle

/status
- Bot latency, servers, users, python version, login state

/tutorial
- Shows the bot tutorial / user guide

/leaderboard <type> [position]
- View LanguageNut leaderboards
- Types: class (by class), school (all students), global (worldwide schools)
- position: number of top entries to show (1-50, default 10)

========================
ADMIN COMMANDS (OWNER ONLY)
========================

/restart          - Restart the bot
/shutdown         - Stop the bot
/update           - git pull + restart
/sync             - Sync slash commands (guild + global)
/clear <amount>   - Delete recent messages (1-100)
/purge <amount>   - Bulk delete messages (1-500)
/logs             - Show last 20 log lines
/reload <cog>     - Reload a cog
/eval <code>      - Async python eval sandbox
/online           - Announce ONLINE ping
/offline          - Announce OFFLINE ping
/say <message>    - Make the bot say something
/embed <title> <desc> [color] - Send an embedded message
/userinfo <user>  - Show detailed user info
/serverinfo       - Show detailed server info
/dm <user> <msg>  - DM a user
/nickname <user> [name] - Change a user's nickname
/lockdown <mode>  - Lock or unlock a channel

SYSTEM NOTES
