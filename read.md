# LNut Bot

A Discord bot for automating LanguageNut homework completion with multi-user support, slash commands, and configurable stealth settings.

## Features

- **Multi-account management** — store multiple LanguageNut accounts with encrypted passwords
- **Slash commands** — full Discord application command interface
- **Homework automation** — automatically fetch and complete all pending homework
- **Configurable stealth** — adjustable speed range and accuracy randomization to avoid detection
- **Progress tracking** — live-updating progress bars for homework completion
- **Per-user settings** — customize speed and accuracy per Discord user
